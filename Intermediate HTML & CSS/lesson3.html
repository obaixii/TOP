HTML Tables

HTML Tables allow us to us to create two-dimensional tables made of rows and columns

Table tags

<table/> is the root tag of HTML table that defines the entire table.

<thead/> is used to group the header rows in a table.

<tbody/> is used to groups the main body of the table content. It's used to separate the header and footer content from the primary data.

<tfoot/> is used to group the footer rows of a table. These rows typically contain summary information or totals for the data presented in the table.

<caption/> is used to provide a title or a description for the table. It is typically placed just below the <table/> tag.

<tr> is for defining the table row.

<th/> defines a table header cell.

<td/> is used for regular table data cells

colspan and rowspan attributes: These attributes can be added to <th> or <td> elements to specify how many columns or rows a cell should span, which can be useful for creating table headers or merging cells.
